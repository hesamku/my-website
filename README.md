
# License Manager (Flask)

این پروژه یک پنل ساده برای مدیریت لایسنس است که کدها در `key.json` ذخیره می‌شوند.
ویژگی‌ها:
- ساخت خودکار لایسنس با الگوی `HESAM-KU-VIP-XXXXXXXXXX`
- تنظیم تاریخ انقضا بر حسب روز (1، 7، 30، 365 و ...)
- دکمه تمدید برای هر لایسنس
- نمایش روزهای باقی‌مانده
- تغییر وضعیت فعال/غیرفعال و حذف
- ورود با نام‌کاربری/رمز: `1234` / `1234`

## اجرا محلی
1. نصب: `pip install flask`
2. اجرا: `python app.py`
3. باز کردن: `http://localhost:5000/login`

فایل `key.json` در روت پروژه وجود دارد و می‌توانید آن را مستقیماً در GitHub قرار دهید. سرور با نوشتن در همین فایل تغییرات را ذخیره می‌کند.
